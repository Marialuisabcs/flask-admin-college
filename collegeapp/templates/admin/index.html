Bem-vindo!

Como utilizar a ferramenta:

Para consultar os dados, basta selecionar no menu acima a tabela que deseja checar. Ao escolher a tabela, você pode criar algum dado em 'create', ou deletar, ao clicar no ícone de lixeira. Para editar o dado armazenado, basta selecionar o ícone do lápis ao lado da lixeira. Algumas ressalvas para a criação de registros:
✿ Um curso não existe sem um coordenador que, por sua vez, é um professor;
✿ Um aluno precisa estar vinculado a um curso já existente;
✿ Um aluno só pode ser matriculado em uma matéria já registrada;
✿ Uma matéria só pode ser lecionada por um professor já cadastrado;

Tecnologias utilizadas:

Framework -> Flask-admin : Microframework baseado em Python com o objetivo de criar páginas web. (Versão: 2.0.1)
ORM -> SQLAlchemy : Object-Relational Mapping (ORM) de código aberto desenvolvido para a linguagem Python distribuído sobre a licença MIT. (Versão: 1.4.23)
SGBD -> SQLite3 : Sistema Gerenciador de Banco de Dados (SGBD) que manipula os dados inseridos no nosso sistema. (Versão: 3.36.0)
Linguagem -> Python : Linguagem de programação de alto nível.
(Versão: 3.9)
